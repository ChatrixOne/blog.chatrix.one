## About Chatrix.One

[Chatrix.One](https://chatrix.one) is a free, decentralized, privacy-friendly XMPP server based in Switzerland. It offers end-to-end encrypted instant messaging with no telemetry, no ads and no personal data required at registration.

- **Protocol:** XMPP (Extensible Messaging and Presence Protocol)
- **Encryption:** OMEMO, OpenPGP, DTLS-SRTP for audio/video
- **Hosting:** Switzerland, powered by 100% renewable energy
- **Platform:** Ejabberd (open source)
- **Compliance:** 100% XEP compliant, XMPP Providers Category A

## About This Repository

This is the source code for the [Chatrix.One Blog](https://blog.chatrix.one) — a Hugo-based static site hosted on GitHub Pages.

The blog covers topics related to XMPP, privacy, open source software, security and self-hosting.
